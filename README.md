# 🏭 Projeto Produção de Fábrica

Sistema desenvolvido em Java para simular e gerenciar o processo de produção de uma fábrica, com foco na aplicação de conceitos de Programação Orientada a Objetos (POO).  
Projeto de caráter acadêmico, voltado para a prática de modelagem de classes, controle de produção e organização de dados.

---

## 🚀 Funcionalidades

- Cadastro de produtos  
- Controle de produção da fábrica  
- Gerenciamento de estoque  
- Simulação do fluxo produtivo   

---

## 🛠️ Tecnologias Utilizadas

- Java (JDK 11)  
- Maven  
- Programação Orientada a Objetos (POO)  

---

## ▶️ Como Executar o Projeto

Execute a classe principal:

```java
org/fabrica/test/Main.java
```

📌 Observações
Certifique-se de ter o Java (JDK 11) instalado.
Utilize o Maven para gerenciamento de dependências.
